Example CKEditor (4.5.3) - with Free Image Browser and Uploader Addon

A basic and simple example with CKEditor. Click on the image: in editor, to test the Image Browse and Upload Addons.
- You can upload images and aufio files (MP3, OGG, WAV), they will be automatically added in CKEDITOR content.

Instructions

• FOR UPLOAD PLUGIN: Open the "plugins/iaupload.php" file and change the values of the $upload_dir array (Lines: 6, 7), if it is necessary to replace the default path "/ckeup/" with the path of the directory for Upload on your server, RELATIVE TO THE ROOT OF YOUR WEBSITE ON SERVER.

• By default, if the uploaded file name exists in the folder for upload, it will be renamed with "filename_NR.ext" (NR is a number). If you want to Overwrite the existing file, set value of 0 to RENAME_F (line 28).

• FOR IMAGE BROWSE PLUGIN: Open the "plugins/imgbrowse/imgbrowse.php" file and change the value of $root property (Line 6), if it is necessary to replace the default path to the folder "/ckeup/" with the path of images directory on your server, RELATIVE TO THE ROOT OF YOUR WEBSITE ON SERVER.

• Sets CHMOD writable permision (0777) to the folders for Images and Audio files on your server (if on Linux system), to allow PHP to upload the files.

• Copy the ckeditor directory (with all its content) on your server.

• To use this CKEditor with Uploader addon in your web site, specify this parameter in your page for CKEditor configuration:   filebrowserImageUploadUrl
- And add this line of code, to allow audio tag in CKEditor content, if you upload Audio file:
CKEDITOR.config.extraAllowedContent = 'audio[*]{*}';

• The imgbrowse plugin is pre installed, to use it in your pages with this CKEditor, specify this parameter in your page in CKEditor configuration:   filebrowserImageBrowseUrl
Example: 
CKEDITOR.replace('content', {
  'extraPlugins': 'doksoft_backup,doksoft_stat,zoom',
  'filebrowserImageBrowseUrl': '/path_to/ckeditor/plugins/imgbrowse/imgbrowse.html?imgroot=PATH_TO_IMAGE_FOLDER',
  'filebrowserImageUploadUrl': '/path_to/ckeditor/plugins/iaupload.php',
});
CKEDITOR.config.extraAllowedContent = 'audio[*]{*}';  //to allow audio tag
The 'extraPlugins': 'doksoft_backup,doksoft_stat,zoom', enable other extra plugins installed in this CKEditor.

- The path to the image folder (PATH_TO_IMAGE_FOLDER) is necessary ONLY IF YOU NOT SET THIS VALUE in "imgbrowse.php". This value MUST BE RELATIVE TO THE ROOT OF YOUR WEBSITE ON SERVER, for example: "imgroot=/your_img_dir".

- You can set the path to the folder with images in the "plugins/imgbrowse/imgbrowse.php" file, to the $root property (Line 6). In this case you NOT NEED TO ADD the ?imgroot=PATH_TO_IMAGE_FOLDER in the "filebrowserImageBrowseUrl".
